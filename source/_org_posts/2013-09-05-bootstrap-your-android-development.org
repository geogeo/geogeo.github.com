** 什么是 android bootstrap
[[http://www.androidbootstrap.com/images/ab-screenshot.png]]
Android Bootstrap 其实是一堆框架的集合, 让你迅速搭好android 开发的基本
框架. 里面包括
- Fragments
- Account Manager
- android-maven-plugin
- Dagger
- ActionBarSherlock
- Menu Drawer
- Robotium
- Integrattion testing
- Parse API
